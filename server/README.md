# rest-api-todo

* **BASE URL**

  http://localhost:3000

# Todo #

**create**
----

* **URL**

  /todos

* **Method:**

  `POST`

* **Success Response:**

  * **Code:** 200 <br />
    **Content:** `{
        "id": <id>,
        "title": <title>,
        "description": <description>,
        "status": <status>,
        "due_date": <due_date>,
        "createdAt": <due_date>,
        "updatedAt": <due_date>
    }`
 
* **Error Response:**

   **Code:** 500 ERROR <br />
    **Content:** `{ error : "" }`
